# 💱 Currency Tracker Application

A real-time Python desktop application that tracks currency exchange rates and converts currencies instantly using a user-friendly Graphical User Interface (GUI).

---

## 🚀 Features

* **Real-time Rates:** Fetches live and accurate currency exchange data.
* **Instant Conversion:** Convert between major global currencies seamlessly.
* **Lightweight & Fast:** Minimal dependencies and quick response time.

---

## 🛠️ Technologies Used

* **Python 3.x** - Core programming language.
* **Tkinter** - For rendering the Graphical User Interface.
* **Requests / API** - To fetch up-to-date financial data.
